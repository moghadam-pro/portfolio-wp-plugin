A structured portfolio manager with reusable card styles, Elementor integration, archives, shortcodes, and Rank Math support.

== Description ==

MPRO Portfolio adds a complete Portfolio content type with structured project metadata, a separate Cover Image, Categories, Tags, three built-in card styles, a single template, archives, Elementor support, and Rank Math integration.

The admin menu is controlled from one source to prevent duplicated taxonomy submenus.

== Installation ==

1. Upload and activate the plugin.
2. Open Portfolio > Styles.
3. Add portfolio items.
4. Use the archive, shortcode, PHP helpers, or Elementor widget.
